
# Wildfire Risk Prediction & Analytics System

Machine Learning and data analytics project focused on wildfire risk estimation using environmental and meteorological conditions.

---

## Project Overview

This project combines:
- Machine Learning
- Interactive Data Visualization
- SQL Analysis
- Streamlit Web Application

to analyze wildfire patterns and predict wildfire risk levels based on weather and forest conditions.

---

## Tools & Technologies

- Python
- Pandas
- Scikit-learn
- Streamlit
- Tableau
- SQLite
- Jupyter Notebook

---

## Project Components

### 1. Data Processing & EDA
- Data cleaning and preprocessing
- Feature engineering
- Exploratory Data Analysis (EDA)

### 2. Machine Learning
- Trained an MLP Classifier model
- Feature scaling and label encoding
- Wildfire risk prediction system

### 3. Streamlit Application
Interactive web application for wildfire risk estimation based on user inputs.

### 4. Tableau Dashboard
Dashboard visualizing:
- Wildfire risk distribution
- Temperature patterns
- Humidity analysis
- Geographic wildfire activity

### 5. SQL Analysis
SQL queries used for:
- Risk-level aggregation
- Environmental condition analysis
- Wildfire severity insights

---

## Key Insights

- Low-risk wildfire cases dominate the dataset
- Higher temperatures are associated with increased wildfire severity
- Lower humidity levels contribute to higher wildfire risk
- Wind speed influences wildfire spread intensity
